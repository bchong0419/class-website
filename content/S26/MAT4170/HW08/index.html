Choice Exercise 5 [4 + 3 + 3] (Group actions)

We defined a (left) group action of the group \((G,*)\) on the set \(X\) as a function \(\cdot\colon G \times X \to X\) such that for all \(x \in X\) and \(g_1, g_2 \in G\)

  • \(e \cdot x = x\), and
  • \(g_1 \cdot (g_2 \cdot x) = (g_1 * g_2) \cdot x\).
  1. [4] For each of the following subparts, give an example of a group \((G,*)\), a set \(X\), and a function \(\cdot\colon G \times X \to X\) such that
    1. The function is a group action (i.e. it satisfies both of the above properties.)
    2. The function satisfies the first property, but not the second.
    3. The function satisfies the second property, but not the first.
    4. The function satisfies neither property.
  2. [3] Prove that if \( G \) is a group that acts on a set \( X \), then the set \[ H = \{g \in G \mid g \cdot x = x \text{ for all } x \in X\} \] is a normal subgroup of \( G \).
  3. [3] Let \( G = \mathbb{Z} \), \( X = (H, *) \), and \( h \in H \) be an element of \( H \). Prove that the map \(n \cdot h = h^n * h * h^{-n}\) is a group action.
