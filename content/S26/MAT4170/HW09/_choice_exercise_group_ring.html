In class we let $G = D_3$ be the dihedral group of the triangle, and did a warmup with the group ring $\mathbb{C}[G]$, where each element is written as a "formal sum" \(\sum_{g \in G} z_g g\) where $z_g \in \mathbb{C}$ is a complex number.

Recall that we add these elements the same way we add polynomials, where we combine like terms. When we multiply these elements, it's like multiplying polynomials, but where we "multiply" the elements of $G$ using the binary operation (and order matters!) In this problem let $x,y,z \in \mathbb{C}[G]$ be given by \[ \begin{align*} x = 3r - 2f, \quad y = -1e + 1 r^2f, \quad z = 1e + 2f \end{align*} \]

  1. Compute $x + y$ and $x + y + z$.
  2. Compute $z \times x$ and $z \times y$.
  3. Compute $z \times (x + y)$
  4. Compute $z \times x + z \times y$.
  5. Since $\mathbb{C}[G]$ is a ring, $(\mathbb{C}[G],+)$ is an abelian group. What is the additive identity $0_R$ in $R$? What is the inverse of the element \(\sum_{g \in G} z_g g\)?
  6. Since $\mathbb{C}[G]$ is a ring, $(\mathbb{C}[G], \times)$ has a multiplicative identity. What is the multiplicative identity $1_R$ in $R$? Prove that is the multiplicative identity by computing $1 \times (\sum_{g \in G} z_g g)$.
  7. Is there an element $w \in \mathbb{C}[G]$ such that $w \times 3r = 1_R \in R$?
  8. Is there an element $\alpha \in \mathbb{C}[G]$ such that $\alpha \times x = 1_R \in R$? (Set a timer for 5 minutes and try to find one. If you find one, what is it? If you don't find one, do you think that $x$ has a multiplicative inverse?)
